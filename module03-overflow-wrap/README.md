## Two tracks (pick one or both)

### Track A — Workbook (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and work the paper / sketch prompts.
2. Complete [CHECKLIST.md](CHECKLIST.md) without relying only on the UI.
3. Optional self-check: `./scripts/module.sh 03 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/overflow-wrap/index.html](http://127.0.0.1:8080/tools/overflow-wrap/index.html)
2. Live: [https://universal-verification-methodology.github.io/learning/tools/overflow-wrap/](https://universal-verification-methodology.github.io/learning/tools/overflow-wrap/)
3. Load the **starter example**, then work challenges.
4. Check off the Track B items in [CHECKLIST.md](CHECKLIST.md).
